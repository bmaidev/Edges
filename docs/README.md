## The shape of it (90 seconds)

1. An **admin** opens `/admin`, creates a **room**, and gets three passcodes
   (admin / facilitator / co-host) plus the join, host, and projector URLs —
   shown **once**.
2. **Participants** open the join URL on their phones (`/r/<room>`) and pick a
   handle (or stay anonymous).
3. The **facilitator** opens `/r/<room>/host`, picks a template (or builds a
   custom session), and drives it phase by phase. The room watches the
   **projector** at `/r/<room>/screen`.
4. A **session** is a sequence of **module** phases. Each module is one tool —
   capture, a poll, a dot-vote, a World Café round, an AI devil's-advocate, etc.
5. At the end, **Archive** wipes the live data and (if an AI key is set) saves a
   synthesised report for the admin.

Nothing is kept beyond 24 hours, and raw contributions are facilitator-only.
